P.A.T.H 계정 및 데이터 삭제 안내

앱 이름: P.A.T.H | 시행일: 2026-03-12 | 버전: 2026-03-12

이 페이지는 Google Play 스토어 등록정보에 표시되는 계정 삭제 요청 안내 페이지입니다. 사용자는 아래 절차에 따라 계정 삭제를 요청할 수 있습니다.

1. 계정 삭제 요청 방법

  1. 로그인 페이지에서 계정에 로그인합니다. 로그인 불가 시 비밀번호 재설정 기능으로 먼저 계정 접근을 복구합니다.
  2. 공부 페이지로 이동한 뒤, 하단 문의 (SUPPORT) 영역에서 관리자 문의를 작성합니다.
  3. 문의 제목 또는 본문 첫 줄에 [계정삭제요청]을 포함하고, 본인 확인을 위해 닉네임/가입 대학 정보를 함께 입력합니다.
  4. 운영팀 본인 확인 후 계정 삭제를 처리하며, 처리 완료 시 서비스 내 메시지로 결과를 안내합니다.

2. 삭제 또는 보관되는 데이터

데이터 유형 처리 방식 추가 보관 기간
계정 기본 정보 (실명, 닉네임, 대학, 프로필) 삭제 요청 승인 후 삭제 즉시 삭제 (시스템 반영 지연 최대 7일)
인증 관련 정보 (비밀번호 해시, 전화번호 해시, 인증 시각) 삭제 요청 승인 후 삭제 즉시 삭제 (시스템 반영 지연 최대 7일)
학습/커뮤니티 업로드 데이터 (성적/GPA 이미지, 게시/활동 기록) 삭제 요청 승인 후 삭제 또는 비식별 처리 즉시 삭제 (백업 잔존분 최대 30일)
접속/보안 로그 (IP, 접속 시간, 보안 이벤트) 서비스 보안 및 부정 이용 대응 목적 보관 후 파기 최대 90일
법령상 보관 의무 데이터 관련 법령 범위 내 별도 분리 보관 후 파기 관련 법령에서 정한 기간

3. 처리 기한

삭제 요청 접수 후 본인 확인이 완료되면 특별한 사정이 없는 한 7일 이내 처리합니다.

4. 유의사항

계정 삭제가 완료되면 복구가 제한될 수 있으며, 보유 자산/학습 기록/커뮤니티 활동 내역은 복원되지 않을 수 있습니다.
